🌦️ Weather Forecast App

A simple and responsive Weather Forecast Web Application that provides real-time weather updates based on user location.

🚀 Live Demo

👉 https://pujalokku49-stack.github.io/Weather-forecast-app/

📌 Features
🌡️ Real-time weather information
📍 Location-based weather search
📊 Clean and user-friendly interface
⚡ Fast and responsive design
🛠️ Tech Stack
HTML
CSS
JavaScript
Weather API

⚙️ How It Works
User enters a city name or uses current location
App fetches weather data from API
Displays temperature, conditions, and other details
